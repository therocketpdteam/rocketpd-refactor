# About Page Members

The About page people sections are now driven by the `member` custom post type.

Create one Member post per person in WordPress Admin. The post title is the display name. Set `Member Type` to place the card in the correct About page section.

## Card Fields by Type

Founder cards:
- Name: use the post title
- Title
- Tagline
- Bio

Team cards:
- Name: use the post title
- Title

Board cards:
- Name: use the post title
- Title
- Association

Shared optional fields:
- Initials
- Sort Order
- Card Gradient
- Headshot

`Legacy Role` is retained only as a fallback for existing staging content. Use `Title` for new entries. Social fields may remain available, but the approved About page card design does not render social links.
